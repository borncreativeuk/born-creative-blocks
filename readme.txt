== Description ==
Various utility blocks for the wordpress editor


== Installation ==

1. Add the plugin from the WordPress plugin directory.
1. Activate the plugin.
1. Navigate to any content area (eg a post or a page)
1. Click the 'add block' button
1. Select the block you wish to add
